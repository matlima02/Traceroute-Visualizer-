# traceroute
Just a simple traceroute in python.
Easy to follow and understand :)

## Installation

Simply clone the repository to your local machine:
```bash
git clone https://github.com/matlima02/traceroute.git
```

## Usage
```bash
cd traceroute
sudo python3 traceroute.py <hostname>
```
